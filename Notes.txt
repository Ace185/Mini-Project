Basic Terminal Commands :-
1. cd -> Change Directory
2. ls -> List of files or folders
3.clear -> Clear Terminal
4. -a -> Shows Hidden Files

Basic Git Commands :-
1. Clone -> Clones the Repository from GitHub to PC or vice cersa.
2. status -> Shows the current status of all files in folder
3. add -> adds the changes to GitHub
4. commits -> commits the changelaptops to GitHub
5. push -> upload local Repository content to remote Repository

Facts :-
1. Local means Repositories on our pc or laptop
2. Remote means Repositories on our GitHub
